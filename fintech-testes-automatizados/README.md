# POC | Automação de Testes de Qualidade de Dados em DBT para um Contexto de Fintech
Esta POC foi estruturada a partir de um fluxo de laboratório adaptado para um cenário de fintech. A proposta usa Docker, SQLite e dbt para simular um pipeline simples de transformação de dados com foco em qualidade, rastreabilidade e automação de testes.
