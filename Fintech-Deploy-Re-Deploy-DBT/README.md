# POC de Modelagem Analítica em dbt para Movimentação Financeira Diária

POC em dbt para modelagem analítica de movimentação financeira diária, com camada intermediária, mart de negócio, testes e documentação técnica. O projeto demonstra a organização de dados transacionais em modelos mais confiáveis e rastreáveis para análise.
