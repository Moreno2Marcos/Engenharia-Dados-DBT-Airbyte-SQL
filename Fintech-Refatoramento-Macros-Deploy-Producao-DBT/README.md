# POC Fintech com dbt: Macros, Refatoramento, Testes e Deploy em Produção
Esta POC demonstra um fluxo de Analytics Engineering em uma fintech usando dbt para organizar transformações de dados, aplicar refatoração com macros, validar regras de negócio com testes e publicar documentação técnica dos modelos. O projeto utiliza Docker, dbt Core, dbt SQLite e bancos separados para desenvolvimento e produção.
